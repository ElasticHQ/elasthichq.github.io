Features Overview (Nerd-Friendly Feature List can be viewed here.)

Real-Time Data

Status information is presented as-it-happens, across any of your clusters. Drill down to any specific Index or Nodes for an in-depth view of all pertinent metrics.


Monitoring Clusters

Stay on top of your cluster health in real-time with top-level overview data and drill-down fine-grained stats.


Multi-Cluster Aware

ElasticHQ connects to all of your clusters, regardless of Elasticsearch version, and allows you to monitor and manage them all from one interface.


Indices Management

Refresh, Optimize, Create, and View Indices and their Metrics with a simple click of a button... Shard and Alias information included.


REST API UI

Stop fighting cURL, REST APIs, and JSON formats. The powerful REST UI does it all for you, with single-click actions.


Version Agnostic

Supports Elasticsearch versions 2.x, 5.x, 6.x, and whatever the latest branch may be in the works.


Diagnostics Checkup

ElasticHQ analyzes key metrics across all of your nodes, and actively provides helpful hints and solutions.


Device Friendly

ElasticHQ looks and behaves great in the latest desktop browsers as well as in tablet and smartphone browsers.


Always On

Even if some of your clusters go down, ElasticHQ keeps on ticking. It is designed to run 24/7 standalone and not dependent on the health of the clusters it monitors.


Managing Type Mappings

View the structure of your documents in one easy-to-read user interface.


Node Monitoring

Actively monitor individual nodes for changes in performance from I/O to Memory usage. It's all here.
