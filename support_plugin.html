ElasticHQ Plugin Installation Instructions

  1. Navigate to your /bin directory and type: plugin -install royrusso/elasticsearch-HQ
    Warning! Remember to check for compatible versions and install the correct one. More Info
  2. Point your browser to /domain:port/_plugin/hq/
  3. The cluster location url field should already be filled in with your server address. Simply hit the connect button to initiate connection.
  4. Optional: You can always verify the installation, by navigating to /elasticsearchinstalldirectory/plugins/ and verifying there is an hq directory.
Tip: Case is important. Older version will be located at /HQ, where newer version reside in /hq.
